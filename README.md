# 기능설명
갤러리의 사진을 누르면 새 액티비티가 실행되어 전체화면으로 누른 사진을 볼 수 있다. 또한 전체화면 상태에서 드래그로 줌인/줌아웃이 가능하다.
아래의 + 버튼을 누르면 새 사진을 추가할 수 있으며 다중선택이 가능하다
